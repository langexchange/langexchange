# Run project

Clone project

```bash
git clone https://github.com/nnhutan/langexchange.git
cd langexchange
```

Run on local environment

```bash
# ./langexchange

yarn install
yarn start
```

Run on docker

```bash
# ./langexchange

bin/setup.sh
```
